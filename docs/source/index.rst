Flask OAuth2 for devices documentation
======================================

.. topic:: Abstract

   Flask-oauth2-devices is an extension to Flask that helps you to create the 
   device flow for OAuth2 providers. It is based on the implementation provided 
   by `Google <https://developers.google.com/accounts/docs/OAuth2ForDevices>`__
   and `Section 3.7 of the OAuth 2.05 specification <http://tools.ietf.org/html/draft-ietf-oauth-v2-05#section-3.7>`__.
